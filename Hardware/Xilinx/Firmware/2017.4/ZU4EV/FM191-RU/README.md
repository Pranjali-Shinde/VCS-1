# VCS-1 FM191 Firmware
This repository contains the required files to generate a hardware design for EMC2-DP using TE0820-4EV + FM191-RU, and generate a linux kernel.

Software used:
* Vivado 17.4
* SDK 17.4
* Petalinux 17.4 (Running on Ubuntu 18.04)

The instructions can be found [here](https://github.com/SundanceMultiprocessorTechnology/VCS-1/wiki/Build-Firmware).
